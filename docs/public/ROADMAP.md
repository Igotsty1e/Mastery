# Mastery Roadmap

## Current phase

MVP hardening and staged migration from a lesson-centric runtime toward a fuller learning engine.

## Near-term work

- wire the shipped backend auth and session model into the Flutter client
- continue migrating learner state from device-local storage to the server
- expand lesson and exercise coverage without weakening deterministic evaluation
- improve public-facing documentation and repository hygiene

## Medium-term work

- deepen the learning-engine layer in controlled waves
- improve dashboard and learner transparency surfaces
- expand audio and richer exercise support where it fits the existing evaluation model

## Deliberate constraints

- keep AI usage narrow and auditable
- avoid replacing deterministic grading with free-form model scoring
- do not overstate the current adaptive capabilities before the migration is complete
