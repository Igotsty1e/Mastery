Eight core screens

Two home states, lesson intro, three exercise types (multiple choice + fill-the-blank + sentence correction), result, and summary. Tokens, layout, components and copy all map 1:1 to the Flutter `ThemeData`, `MasteryButton`, `MasteryCard`, `ResultPanel`, `SectionEyebrow` widgets called for in DESIGN.md.

01 · Home

Onboarding

Editorial wordmark sells the philosophy. Three calm proof points. One CTA. Replaces the current default Material 3 onboarding.

02 · Home

Dashboard

Level chips with locked states. Single hero progress card with the live lesson, fraction in IBM Plex Mono. "Coming next" stays quiet.

03 · Lesson

Lesson Intro

B2 pill, Fraunces title, three rule cards (Use / Form / Important) with distinct tints. Examples block in primary-soft. Ends with one CTA.

Active state

Result revealed

  • Google Fonts → self-hosted assets in app/assets/fonts
  • SVG icons → Material outlined or custom
