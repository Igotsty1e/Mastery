Onboarding · 3 directions

Three takes on the locked 2-step onboarding (Promise → Assembly) plus the dashboard-as-home and quieter first exercise. All three sit inside the existing Quiet Premium Coach aesthetic — same dusty-rose anchor, same Fraunces + Manrope, same paper background. They differ in density, ornamentation, and pacing.

Source spec: docs/plans/arrival-ritual.md. Tokens: docs/design-mockups/tokens.css. Pick a direction, then we replace app/lib/screens/onboarding_arrival_ritual_screen.dart and the dashboard-as-home contract.

What's the same across all three

Tokens, fonts, palette, copy. Each direction shows the same 4 screens with the same content — the difference is purely visual hierarchy and ornamentation density.

What's different

A = editorial restraint. B = editorial + linework warmth. C = reductive whitespace and modern signage.

How to evaluate

On mobile, scroll all four tiles in each direction. Ask: which one would I want to see every time I open the app? Which dashboard feels like \u201chome\u201d? Which first exercise feels least like a quiz and most like work?
